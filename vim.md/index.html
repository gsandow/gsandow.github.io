 33 | w  | 移动到单词开头| e  | 移动到单词结尾|

 47 以什么样的形式进入插入模式
 48

 55 区域选择
 56 
 57 其格式为 `<action>a<object>` 和 `<action>i<object>`
 58 
 59 - action可以是任何的命令，如 d (删除), y (拷贝), v (可以视模式选择)。
 60 - object 可能是： w 一个单词， W 一个以空格为分隔的单词， s 一个句字， p 一个段落。也可以是一个特别的字符："、 '、 )、 }、 ]。
 61 
 62 举例： `daw` 删除一个单词
 63 
 64 块操作
 65 
 66 命令模式下按 ctrl + v 移动光标选中需要修改的行（区域），
 67 然后按x 或者 d 删除该内容
 68 
 69 批量在行首增加内容：
 70  `ctrl + v` >> `0` >> `I` >> 写下想增加的内容 >>`Esc`
 71 
 72 ![rectangular-blocks](/images/rectangular-blocks.gif)
 73 
 74 批量在行尾增加内容：
 75 
 76  `ctrl + v` >> `$` >> `A` >> 写下想增加的内容 >>`Esc`
 77 
 78 块移动  `<` , `>` 左右缩进 。 `=` 自动给缩进。
 79 
 80 自动补全
 81 
 82 vim 下在编辑模式下也有自动补全功能便不是tab键，而是 `ctrl + n`， `ctrl + p` 如果匹配到多个可以按多次来找到自己想要的单词
 83 
 84 宏录制
 85 
 86 `qa`  开始录制宏，并把宏的名字取名为a
 87 `q`  停止录制宏
 88 `@a`  用@来调用宏并执行
 89 `@@`  快捷的执行最新操作的宏
 90 
 91 ![macros](/images/macros.gif)

 93 `qa`  >>` Y` >> `p` >> `ctrl + a` >> ` q` >> ` @a` >> ` 100@@`
